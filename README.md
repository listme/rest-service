# webserver

Servicio REST de gestión de listas

# API

El API se puede obtener como colección [postman][1] en [este link][2]

## `GET /item/:id`

Devuelve el contenido del item `:id`

## `POST /item`

Inserta un nuevo item

## `PUT /item/:id`

Actualiza el contenido del item `:id`

## `DELETE /item/:id`

Elimina el item `:id`
